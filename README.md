
# Calculadora
Este é um projeto simples de uma calculadora que realiza operações matemáticas básicas, desenvolvido como uma aplicação web.

### Funcionalidades
- Adição
- Subtração
- Multiplicação
- Divisão

### Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
